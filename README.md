# ⚡ AI-Powered Information Extraction (Invoices)

A sophisticated automation workflow built with **n8n** and **AI Agents** to transform messy invoice documents into structured, actionable data.

## 🚀 Overview
Manual invoice processing is slow and prone to error. This project automates the entire lifecycle of an invoice—from ingestion to data extraction—using LLMs (Large Language Models) to understand context regardless of the invoice layout.

## ✨ Key Features
* **Intelligent Extraction:** Uses AI Agents to identify Invoice ID, Date, Tax, and Line Items.
* **Seamless Integration:** Connects multiple nodes to structure data for ERP/Databases.
* **High Precision:** Handles various formats (PDF, Images) with high accuracy.

## 🛠️ Tech Stack
* **Automation:** [n8n](https://n8n.io/)
* **AI:** OpenAI / Anthropic (AI Agent Node)
* **Logic:** JavaScript / JSON

## 📂 How to Use
1. Download the `invoice-extraction-workflow.json` file from this repo.
2. Import it into your n8n instance.
3. Configure your AI credentials.
4. Run and enjoy the automation!
